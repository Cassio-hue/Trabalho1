\doxysubsection{Detailed Description}
Domínio \mbox{\hyperlink{class_nome}{Nome}}. 

Possui uma função validar. Se o nome (tipo string) passado para a função for válido, o nome é armazenado. Caso contrário, ocorre o lançamento de uma exceção.

{\bfseries{Regras para o nome ser aceito}}\+:
\begin{DoxyEnumerate}
\item Até 30 caracteres
\item Formato\+: prenome sobrenome
\item Não pode ter dois espaços em branco
\item Não pode terminar ou começar com espaço em branco
\item Somente letras
\item As letras iniciais do prenome e sobrenome devem ser maiúsculas
\end{DoxyEnumerate}

{\bfseries{Exemplos de nomes válidos}}\+:
\begin{DoxyItemize}
\item Cassio Vinicius
\item Joaquim Silva
\item Dom Quixote
\item Princesa Jujuba 
\end{DoxyItemize}
