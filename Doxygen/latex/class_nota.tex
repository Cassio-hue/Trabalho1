\doxysubsection{Detailed Description}
Domínio \mbox{\hyperlink{class_nota}{Nota}}. 

Possui uma função validar. Se a nota (tipo string) passada para a função for válida, a nota é armazenada. Caso contrário, ocorre o lançamento de uma exceção.

{\bfseries{Regras para a nota ser aceita}}\+:
\begin{DoxyEnumerate}
\item Somente números
\item A nota deve ser um número entre 0 e 10
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item 0 é a menor nota possível e 10 a maior nota possível 
\end{DoxyItemize}
