\doxysubsection{Detailed Description}
Domínio \mbox{\hyperlink{class_data}{Data}}. 

Possui uma função validar. Se a data (tipo string) passada para a função for válida, a data é armazenada. Caso contrário, ocorre o lançamento de uma exceção.

O formato da data considerado válido\+: {\bfseries{DIA/\+MES}}

{\bfseries{Formato dos dias}}\+: Numeros entre 01 -\/ 31, número menor que 10 deve ser acompanhado com um zero.

{\bfseries{Formato dos meses}}\+:
\begin{DoxyItemize}
\item JAN
\item FEV
\item MAC
\item ABR
\item MAI
\item JUN
\item JUL
\item AGO
\item SET
\item OUT
\item NOV
\item DEZ
\end{DoxyItemize}

A função validar não leva em consideração anos bissextos

{\bfseries{Observações}}\+:
\begin{DoxyItemize}
\item O mês de FEV vai até o dia 29
\item Os meses\+: ABR, JUN, SET, NOV vão até o dia 30
\item Os meses\+: JAN, MAR, MAIO, JUL, AGO, OUT vão até o dia 31 
\end{DoxyItemize}
