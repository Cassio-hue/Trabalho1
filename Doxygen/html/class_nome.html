Detailed Description

Domínio Nome.

É importante se alientar que não ocorre a validação da acentuação, logo caso esteja presente algum acento no valor passado, será lançada uma exceção! Possui uma função validar. Se o nome (tipo string) passado para a função for válido, o nome é armazenado. Caso contrário, ocorre o lançamento de uma exceção.

Regras para o nome ser aceito:

  1. Até 30 caracteres
  2. Formato: prenome sobrenome
  3. Não pode ter dois espaços em branco
  4. Não pode terminar ou começar com espaço em branco
  5. Somente letras
  6. As letras iniciais do prenome e sobrenome devem ser maiúsculas

Exemplos de nomes válidos:

  • Cassio Vinicius
  • Joaquim Silva
  • Dom Quixote
  • Princesa Jujuba
